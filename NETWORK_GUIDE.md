# 🌐 Как подключить друга к мессенджеру

## Быстрый старт

### 1. Запустите мессенджер
```bash
./start_network.sh
```
или
```bash
python3 launcher.py
```

### 2. Узнайте свой IP
Скрипт автоматически покажет ваш локальный IP, например: `192.168.104.160`

### 3. Отправьте другу ссылку
```
http://ВАШ_IP:8000/frontend.html?user=ИмяДруга
```

Пример:
```
http://192.168.104.160:8000/frontend.html?user=Mike
```

## Инструкция для друга

1. **Подключитесь к той же WiFi сети** (тот же роутер)
2. **Откройте браузер** (Chrome, Firefox, Safari)
3. **Вставьте ссылку**, которую вам отправили
4. **Нажмите "+ New Chat"** (кнопка справа вверху)
5. **Выберите собеседника** из списка онлайн пользователей
6. **Начните общаться!** 💬

## Проверка подключения

Если друг не может подключиться:

### Проверьте, что сервер запущен
```bash
ss -tuln | grep -E ':(8000|8765)'
```
Должно показать:
- `0.0.0.0:8765` - WebSocket сервер
- `0.0.0.0:8000` - HTTP сервер

### Проверьте IP адрес
```bash
ip addr show | grep "inet " | grep -v 127.0.0.1
```

### Проверьте брандмауэр (если нужно)
```bash
sudo ufw allow 8000/tcp
sudo ufw allow 8765/tcp
```

### Проверьте подключение с другого устройства
```bash
ping ВАШ_IP
```

## Возможные проблемы

### "Не могу подключиться"
- Убедитесь, что вы в одной WiFi сети
- Проверьте, что сервер запущен
- Попробуйте отключить брандмауэр временно

### "Не вижу друга в списке онлайн"
- Обновите страницу (F5)
- Проверьте, что друг открыл ссылку с другим именем
- Подождите 5 секунд (автообновление списка)

### "Сообщения не отправляются"
- Проверьте консоль браузера (F12)
- Убедитесь, что WebSocket подключен (зеленый индикатор)
- Перезапустите сервер

## Технические детали

- **HTTP сервер**: порт 8000 (веб-интерфейс)
- **WebSocket сервер**: порт 8765 (сообщения в реальном времени)
- **База данных**: SQLite (messenger.db)
- **Хранилище**: localStorage браузера + SQLite на сервере

## Безопасность

⚠️ **Важно**: Этот мессенджер работает только в локальной сети!
- Не доступен из интернета
- Нет шифрования (не используйте для конфиденциальной информации)
- Подходит для домашней сети, офиса, учебного класса

## Дополнительные возможности

### Добавить больше пользователей
Просто откройте новую вкладку с другим именем:
```
http://ВАШ_IP:8000/frontend.html?user=Charlie
http://ВАШ_IP:8000/frontend.html?user=David
```

### Просмотр истории сообщений
Все сообщения сохраняются в `messenger.db`:
```bash
sqlite3 messenger.db "SELECT * FROM messages ORDER BY id DESC LIMIT 10;"
```

### Очистка истории
```bash
rm messenger.db
# База будет создана заново при следующем запуске
```
